PROJECT OVERVIEW
----------------
AI-Powered Digital Call Center is a modern, intelligent customer care platform built using Spring Boot, Spring AI, and React.
It provides automated AI chat support, ticket management, analytics, and a structured knowledge base.

KEY OBJECTIVES
--------------
- 24/7 automated customer support
- Reduced response time
- Improved customer satisfaction
- Centralized conversation & ticket management
- Context-aware AI with memory
- Searchable policy-based knowledge base

FEATURES
--------
Authentication:
- Login & Signup
- Session handling
- Protected routes

Dashboard:
- Total conversations
- Customer satisfaction
- Average response time
- Active chats
- Quick actions

Chat Interface:
- AI-powered real-time chat
- Conversation sidebar
- Streaming responses
- Conversation memory

Knowledge Base:
- Extracted company policies

- Searchable content

Analytics:
- Performance metrics
